# 🌾 Crop Recommendation System

An end-to-end Machine Learning web application that recommends suitable crops based on soil nutrients and climatic conditions.

## 🚀 Features
- Predicts crop type using NPK values and environmental factors
- Flask-based web interface
- Trained ML classification model
- User-friendly UI

## 🧠 Tech Stack
- Python
- Machine Learning (Scikit-learn)
- Flask
- HTML, CSS
- Git & GitHub

## 📊 Input Parameters
- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- pH
- Rainfall

## ▶️ How to Run
```bash
pip install -r requirements.txt
python app.py
